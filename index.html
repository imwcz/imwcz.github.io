Solution

  1. if x = 0, then maximum of ax^2+bx equals 0;
  2. if x > 0, then ax^2+bx is maximum when ax+b is maximum;
  3. if x < 0, then ax^2+bx is maximum when ax+b is minimum;
  4. and we find that when a_1\ge a_2 and b_1\ge b_2 then a_1x+b_1 always morn that a_2x+b_2;
  5. so we can sort all the lines, and delete the lines that is no value;
  6. the lines after the Sort content that a is increased and b is decreased;
  7. then we solve Upper convex shell so that the maximum values of all functions are on it.
  8. if x less than zero, then we can change all b to -b, then change x to -x, so we can solve it like x more than zero.

SP1043 GSS1

Posted on 2018-09-01 | In 信息学竞赛 , 题目 |

Description

You are given a sequence A[1], A[2], …, A[N] . ( |A[i]| ≤ 15007 , 1 ≤ N ≤ 50000 ). A query is defined as follows:
Query(x,y) = Max { a[i]+a[i+1]+…+a[j] ; x ≤ i ≤ j ≤ y }.
Given M queries, your program must output the results of these queries.

Solution

用线段树维护区间和, 区间最大子段和, 区间左侧最大和, 区间右侧最大和.
最重要的部分是合并两个区间.

其中卡过时间限制花了很长时间;

  1. 如果按照常规思路来查询, 每次递归查询都需要至少合并一次区间;
  2. 使用new, delete来管理内存可能会出现严重的内存泄露, 而且效率不高;
  3. 合并区间中有大量的无用步骤, 需要精简;

解决方法:

  1. 更改查询方式, 使得每次至多合并一次区间;
  2. 直接使用局部变量, 并且在合并时用值代替指针;
  3. 将合并过程精简到一条语句;
  4. 使用读入优化.

Solution
