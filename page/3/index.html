二维例子

$$
\begin{aligned}
\begin{cases}
2x-y=0\\
-x+2y=3
\end{cases}
\end{aligned}
$$
每个方程都代表着三维空间中的一条直线.

行图像

$$
\begin{pmatrix}x\\y
\end{pmatrix}
\begin{pmatrix}
2&-1\\
-1&2
\end{pmatrix}=
\begin{pmatrix}
0\\3
\end{pmatrix}
$$

列图像

$$
x
\begin{pmatrix}
2\-1
\end{pmatrix}+y
\begin{pmatrix}
-1\\2
\end{pmatrix}
=\begin{pmatrix}
0\\3
\end{pmatrix}
$$

三维例子

$$
\begin{cases}
2x-y+0z=0\\
-x+2y-z=-1\\
0x-3y+4z=4
\end{cases}
$$
每个方程都代表着三维空间中的一个平面.

列的线性组合是否能覆盖整个三维空间.

矩阵乘法

$$
\begin{pmatrix}
2 &5\\1 &3
\end{pmatrix}
\begin{pmatrix}
1\\2
\end{pmatrix}
=1\begin{pmatrix}
2\\1
\end{pmatrix}+
2\begin{pmatrix}
5\\3
\end{pmatrix}=
\begin{pmatrix}
12\\7
\end{pmatrix}
$$
或者
$$
\begin{pmatrix}
\begin{pmatrix}2 &5\end{pmatrix}
\cdot
\begin{pmatrix}1\\2\end{pmatrix}
\\
\begin{pmatrix}1 &3\end{pmatrix}
\cdot
\begin{pmatrix}1\\2\end{pmatrix}
\end{pmatrix}
$$

第一题

有一个用这些数去筛其他数的做法, 我想到了, 但是设想极限情况, 这些数分别为
1, 2, …, n, 那么每次筛一遍总共$N/1, N/2, …, N/N$ 次, 感性感觉这个复杂度是$n^2$的, 于是就没写, 实际上这个复杂度是$N\log N$的.

于是写了$N\sqrt N$.

第二题

去年10月是做过这个题的, 全忘了全忘了. 于是只好手动YY新做法, 设区间为$l, r$, 两个字符分别是$A, B$.那么
$$
\max S_{A, r} - S_{A, l - 1} - S_{B, r} + S_{B, l - 1}.
$$
可以将这个式子分成两部分$S_{A, r} - S_{B, r}$和$S_{B, l - 1}, S_{A, l - 1}$.发现这两个式子实际上是一样的.将这两个式子的最大值加起来可以.
如何维护形如$S_{A, i} - S_{B, i}$的式子的最大值呢?将这个式子记为$C_{A, B, i}$

可以$26\times 26$的复杂度维护, 其实这样复杂度就很高了, 不这样搞.

可以发现每次加入一个字符$A$时, 受到影响的只有$S_{A, x, i}$, 这样就可以$O(26N)$的复杂度做掉这个题.

有一个诡异的错误, 如果判B字符出现过再更新, 那么答案会小; 如果不判, 答案会大.
于是就全程De这个bug, 一直到考试结束.

虽然最后还是没调出来, 成功被这个bug卡掉30分.改天重构这个代码.

第三题

于是连Dijkstra的显然做法都没有写, 你看你们写那么长时间拿个20-30分, 我半分钟拿10分诶.

码力太弱.

Solution

  1. if x = 0, then maximum of ax^2+bx equals 0;
  2. if x > 0, then ax^2+bx is maximum when ax+b is maximum;
  3. if x < 0, then ax^2+bx is maximum when ax+b is minimum;
  4. and we find that when a_1\ge a_2 and b_1\ge b_2 then a_1x+b_1 always morn that a_2x+b_2;
  5. so we can sort all the lines, and delete the lines that is no value;
  6. the lines after the Sort content that a is increased and b is decreased;
  7. then we solve Upper convex shell so that the maximum values of all functions are on it.
  8. if x less than zero, then we can change all b to -b, then change x to -x, so we can solve it like x more than zero.

SP1043 GSS1

Posted on 2018-09-01 | In 信息学竞赛 , 题目 |

Description

You are given a sequence A[1], A[2], …, A[N] . ( |A[i]| ≤ 15007 , 1 ≤ N ≤ 50000 ). A query is defined as follows:
Query(x,y) = Max { a[i]+a[i+1]+…+a[j] ; x ≤ i ≤ j ≤ y }.
Given M queries, your program must output the results of these queries.

Solution

用线段树维护区间和, 区间最大子段和, 区间左侧最大和, 区间右侧最大和.
最重要的部分是合并两个区间.

其中卡过时间限制花了很长时间;

  1. 如果按照常规思路来查询, 每次递归查询都需要至少合并一次区间;
  2. 使用new, delete来管理内存可能会出现严重的内存泄露, 而且效率不高;
  3. 合并区间中有大量的无用步骤, 需要精简;

解决方法:

  1. 更改查询方式, 使得每次至多合并一次区间;
  2. 直接使用局部变量, 并且在合并时用值代替指针;
  3. 将合并过程精简到一条语句;
  4. 使用读入优化.
