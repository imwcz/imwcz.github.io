P4159 [SCOI2009]迷路

Posted on 2018-08-10 |

Description

windy在有向图中迷路了。 该有向图有 N 个节点，windy从节点 0 出发，他必须恰好在 T 时刻到达节点 N-1。 现在给出该有向图，你能告诉windy总共有多少种不同的路径吗？ 注意：windy不能在某个节点逗留，且通过某有向边的时间严格为给定的时间。

Solution

对于时间全为1的情况可以这样考虑: $f_i(1, n)$表示经过$i$个时间从1走到n的方案数, 那么$f_i=f_1^i$.

推广一下, 考虑$u$到$v$一条时间为$c$的边, 将每个点拆成9个点, 第二个点向第一个点连边, 第三个点向第二个点连边, ……
在从u的第一个点向v的第c个点连边, 能保证c时间后u和v能直接到达.
