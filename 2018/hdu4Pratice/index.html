这四道题目中只有1394这道需要注意,

Description

求将这个序列的前面任意一段放到后面形成的序列中最小的逆序数.

Solution

先计算出原序列的逆序数, 然后观察将一段放到后面产生了什么影响,

  • 首先把第一个元素$a_0$放到后面, 那么比$a_0$小的数$a_j$与其形成的逆序对$(a_0, a_j)$消失了, 这部分的数量是$a_0$; 然后比$a_0$大的元素又与其形成了逆序对, 数量是$n-1-a_0$.
  • 重复上面的步骤, 可以得到所有能生成的序列, 取最小值即可.
