Mininum Deletions

Solution

　　$n$个数的最大公约数不会因为删去一个数而变小, 因此要么需要删去所有的数要么最大公约数本来就是1.

Xor Again

Solution

　　发现当$i\ne j$时$A_i+A_j$会被算入到Xor中两次, 抵消, 因此答案为$2A_i\oplus \cdots\2A_n$.

Mighty Friend

Solution

　　只需要看后手的分数加上先手比后手大的那些能否获胜即可.

Dibs on Fibs

Solution

　　设$x=A_i,y=B_i$, 那么$x,y$可以看成是参数, 那么$Fib_i=a_ix+b_iy$, 只需要递推$a_i,b_i$, 直接计算得到答案.
　　注意取模.

Code

Fake Binary Search

Solution

　　模拟它的二分查找的过程, 把其中可能会发生取向错误的位置替换掉即可.注意特判不可能的情况.
