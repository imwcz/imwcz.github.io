吐槽

只是校内互测.
而且题目难度倒是真心不大(会做就算难度不大), 就是数据出了很多锅很气人……

题目

第一题

Solution

根据经验, $n$条直线将平面划分的最大个数为$1+1+\cdots+n$.然而这个X型和两条直线没啥区别, 于是答案为
$$1+1+\cdots+2n=1+\frac{2n(2n+1)}{2}=1+n(2n+1)$$

Solution

发现查询一个$k$的复杂度是$O(n)$, 于是便可以暴力查询, $n,k\le 10^6$不会超时吗?不会……只需要判掉完全不可能的情况就好了.

第四题

Solution

标程是树上莫队, 还有一个做法是校内大佬(侧边栏链接第三个…)想出来的, 是听说学长曾讲过类似的东西(我怎么没记得)然后改出来的:

  • 具体是将两个相同颜色的点在其lca上减去一个, 这样直接统计没有重复.
  • 我的做法几乎是卡着时间线过的, 具体是每个节点维护一个set直接暴力合并到父亲节点然后一路合并上去, 据说加上启发式合并会快非常多(大概是几十倍)然而我不会.
  • 树上莫队将树用dfs序生成一个序列, 然后就可以用类似HH的项链的做法做了.

我的做法树的高度很高的时候就挂了(例如链)……所以只能算是暴力做法……
