第二题

Solution

先对每个位置减去$k$, 求一遍前缀和, 那么一段区间$[l,r]$合法需满足$s_r-s_{l-1} \le 0$, 因此令右端点为$r$, 满足条件的左端点$l$需满足$s_{l-1}>s_r$, 于是转化为类似于求逆序对的问题, 用线段树或树状数组都可以.
若$s_r\le 0$, 则区间$[1,r]$也满足情况.

  • 上界$O(N^2)$, 下界$\Omega(N)$的算法在随机数据下可以通过.
  • $O(N\log_2 N)$的算法如线段树, 二分通过看常数.
  • $O(N)$的算法可以正常通过, 双指针.

第二题

Solution

设$a \text{ and } b = b - x, a \text{ or } b = a + x$, 那么$(a-x)^2+(a+x)^2=a^2+b^2+2x(x+a-b)$.
可以看出能进行操作时总是最优的.

观察发现, 两个数$a,b$进行操作相当于是将两个数中不同的数位移动到同一个数上, 因此无需模拟题意, 直接统计数位的个数, 然后尽量往数上加即可.
